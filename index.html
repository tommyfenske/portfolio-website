Jetpack Joyride Demake

I remade the popular mobile game Jetpack Joyride inside the game engine, Pico-8. Pico-8 only allows 16 colors, 4 sound channels, and 256 8x8 sprites, hence the name "De-make". I learned to code using Pico-8's version of Lua. The engine's limitations meant Object Oriented, abstract, and reusable code was a priority. I made all code, sounds, and sprites. This was made as a result of my participation in the 20 Games Challenge.

Web Blackjack

My Blackjack game was made using HTML, CSS, and JavaScript. I learned how to interact with HTML files and elements using JavaScript, create a rudimentary AI for the "dealer", and implement arrays to emulate a real deck of cards to replace generating random numbers for card values. A Chrome Extension version was submitted, reviewed, and successfully published to the Chrome Web Store.

To Do App with Database

Created a web app where users can add items to a "to-do" list. Multiple users can view the same list, and add and delete items on that shared list, which will be updated in real-time. The interface is coded with HTML/CSS, while JavaScript and a Firebase Realtime Database manage the back end. Built with mobile support as the focus.
